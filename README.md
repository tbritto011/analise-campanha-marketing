# analise-campanha-marketing
Otimização de Orçamento de Mídia: Análise de ROI e Lucro Líquido

Objetivo: Analisar um dataset de mais de 200 mil registros de campanhas de marketing para diagnosticar a performance histórica, identificando os canais com maior capacidade de escala e geração de lucro líquido.

Ferramentas: BigQuery (SQL) para processamento, agrupamento e rankeamento via Window Functions; Python (Pandas, Seaborn) para cálculo de retorno financeiro e visualização de dados e Estatística.

Análise do ROI:
Durante a análise, identifiquei uma distinção crítica entre eficiência de capital (ROI) e capacidade de escala (Lucro Absoluto). O canal YouTube em Miami lidera em ROI (5.11), mas o Facebook em Chicago é o responsável pelo maior volume de lucro absoluto (gerando mais de R$ 90 milhões limpos). Isso demonstra que o Facebook possui um Product-Market Fit mais consolidado para grandes volumes de investimento, enquanto o YouTube em Miami apresenta uma oportunidade de alta eficiência marginal que pode ser testada em maior escala.
